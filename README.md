# Google Keep Desktop

## Description

Cross platform (MacOS, Windows, Linux) Electron powered desktop Google Keep application.

## Features

- Smooth scrolling
- Caches locally your account information

## Requirements

- `Node.js` >= 12.13.0
- `npm` >= 6.9.0

## Compiling

#### 1) `npm run bootstrap`

#### 2) Extract binary from generated `./bin` folder and install as ordinary binary
